# PSY6422_Project

# Is Wales wetter than England?

## Overview

This project looks at whether Wales is wetter than England by looking at the annual rainfall recordings (mm) from 1836 to 2024 in England and Wales. 
This analysis is written in R using Quarto to produce a fully reproducable, self-contained HTML report. 

# Key Features
This project has several elements including: 
- Data cleaning and wrangling using tidyverse
- Exploratory analysis of long-term rainfall trends
- Animated visualisation using ggplot2, dyplyr and gganimate
- Reproducible workflow using Quarto for transparent reporting

# Requirements

- R (version 4.5.1)
- R packages: tidyverse, ggplot2, dplyr, gganimate, gifski #required to render GIFS
- Quarto
